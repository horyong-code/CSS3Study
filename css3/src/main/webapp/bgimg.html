background - 배경

  1. background-color: 색상값; - 배경 색상
  2. background-image:url("경로"); - 배경 이미지

background-image 2개 지정

background-image: url(""), url("");

background-image의 속성들

  • background-repeat:repeat/repeat-x/repeat-y/no-repeat; 반복 지정
    1. repeat (기본) 양사방 반복
    2. repeat-x 가로로만 반복
    3. repeat-y 세로로만 반복
    4. no-repeat 반복 안함
  • background-position: ; 위치 지정
    1. left/center/right top/center/bottom
    2. Xn Yn
  • background-size: 크기 지정;
    1. cover - 이미지 하나로 영역을 다 덮기 (이미지 잘려나갈 수도 있음)
    2. contain - 가로나 세로에 맞춰 이미지로 덮기
    3. Xn Yn (px, %, em...) - 가로 세로 순서로 직접 지정
  • background-attachment: /fixed - 스크롤 여부;
    1. scroll - (기본) 스크롤
    2. fixed - 고정
    3. local - 영역에서

background-attachment: ; 스크롤 여부
