margin

border를 기준으로 요소 외부의 양사방 여백

1. margin의 각 방향별 지정 속성

  • margin-:top; 위쪽에만 n만큼 지정
  • margin-:right; 오른쪽에만 n만큼 지정
  • margin-:bottom; 아래쪽에만 n만큼 지정
  • margin-:left; 왼쪽에만 n만큼 지정

2. margin의 값 개수별 지정법

  • margin:n; 양사방 n으로 동일
  • margin:n n; 상하 좌우 순서로 각각 지정
  • margin:n n n; 상 좌우동일 하 순서로 각각 지정
  • margin:n n n n; 상 우 좌 하 순서로 각각 지정

block요소는 인접요소끼리의 상하 margin은 큰 값이 적용됨
inline요소는 상하 margin 적용 안됨.
인접요소끼리의 좌우 margin은 서로 더하기됨.
* block 요소를 항상 정중앙에 배치할 때는
margin: n auto; 적용 (auto = 좌우 margin이 항상 같음)
