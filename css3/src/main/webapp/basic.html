CSS 적용

  • External 방식 - 외부 파일 방식. 스타일 소스가 웹 문서와 별개.
    head안에 <link href="*.css" rel="stylesheet">
    통일성 있는 디자인 가능, 유지 보수 유리
  • Internal 방식 - 내장 방식. 웹문서의 head 안에 스타일 소스가 위치.
    <style type="text/css"> ... </style>
    웹페이지에 종속적. 해당 문서에만 적용됨
  • Inline 방식 - 태그 뒤에 바로 스타일 소스가 위치.
    통일성 있는 디자인 불가능, 유지 보수 불리
    <tag style="...">
    CSS 적용 우선 순위는 가장 높음. body만 수정 가능한 상황에서 활용

CSS 선택자 대표 3가지

  • tag 선택자 - tag 그대로 활용. 강제 & 자동 적용
  • id 선택자 - tag에 id를 추가하여 활용 (#id) id가 중복 적용 불가하므로 스타일도 중복 적용 불가. 주로 구조적 용도로 쓰임.
  • class 선택자 - tag에 class를 추가하여 활용 (.class)
    • class가 중복 적용 가능하므로 스타일도 중복 적용 가능. 주로 디자인적 용도로 쓰임.
