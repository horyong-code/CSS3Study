transform - 변형

transform 변형 함수들

  • transfrom: translate(); - 위치 변형
  • transfrom: scale(); - 크기 변형
  • transfrom: rotate(); - 회전 변형
  • transfrom: skew(); - 기울기 변형

transform-origin: ; 변형

  • left / center / right - top / center / bottom
  • Xn Yn (n은 음수 가능. px, %, em ...)

1. transform: translate(); - 위치 변형

transform: translateX(1000px);

transform: translateX(-500px);

transform: translateY(-250px);

transform: translateY(250px);

transform: translate(1000px, 200px);

transform: translate(100%, 100%);

[center]
1. position: absolute;
left: 50%; top: 50%;
2. transform: translate(-50%, -50%);

2. transform: scale(); - 크기 변형

transform: scaleX(0.3);

transform: scaleX(1.5);

transform: scaleY(0.3);

transform-origin: left bottom;

transform: scaleY(2);

transform: scale(0.5, 0.5);

transform: scale(1.3, 1.3);

3. transform: rotate(); - 회전 변형 (n은 음수 가능)

transform: rotateX(80deg);

transform: rotateY(80deg);

transform: rotateX(180deg);

transform: rotateY(180deg);

transform-orgin: 100% 0;

transform: rotate(-360deg);

transform-origin: 0 0;

transform-origin: 1000px -200px;

transform: rotate(1080deg);

4. transform: skew(ndeg); - 기울기 변형 (n은 음수 가능)

transform: skewX(30deg);

transform: skewX(-30deg);

transform: skewY(30deg);

transform: skewY(-30deg);

transform: skew(30deg, 30deg);

transform: skew(30deg, 30deg);
