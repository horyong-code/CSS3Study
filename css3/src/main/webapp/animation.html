animation의 정의

@keyframes 애니명 { 0% {속성:속성값;} ----- 첫 장면에서의 상태 ... 100% {속성:속성값;} ----- 끝 장면에서의 상태 }

animation의 세부 속성들

  • animation-name (필수) 적용할 애니메이션 이름
  • animation-duration (필수) 진행 시간
  • animation-delay 지연 시간
  • animation-iteration-count 반복 횟수 (무한반복은 infinite)
  • animation-direction 진행 방향
    (normal/reverse/alternate/alternate-reverse 정방향/역순/왕복/역순왕복)
  • animation-timing-function 가속 감속 등
    (ease/ease-in/ease-out/ease-in-out/ linear/cubic-bezier(n,n,n,n) )
  • animation-fill-mode 멈춤 상태일 때의 장면 지정
    (none/forwards/backwards/both/initial)

animation-play-state: running / paused;
