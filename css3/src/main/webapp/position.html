position - 위치 지정 방식

  • position: static; - (기본) 소스 순서대로 정렬
  • position: relative; - 현 위치에서 ~ 거리 지정
  • position: absolute; - 상위 요소를 기준으로 ~ 거리 지정
  • position: fixed; - 브라우저를 기준으로 ~ 고정
  • position: sticky; - relative와 fixed 믹스

거리값 지정

  • 수평 : left / right
  • 수직 : top / bottom

5. position: sticky; relative와 fixed 믹스
