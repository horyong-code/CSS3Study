padding

border를 기준으로 요소 내부의 양 사방 여백

1. padding의 각 방향별 지정 속성

  • padding-top: n; 위쪽에 n만큼 지정
  • padding-right: n; 오른쪽에 n만큼 지정
  • padding-bottom: n; 아랫쪽에 n만큼 지정
  • padding-left: n; 왼쪽에 n만큼 지정

2. padding의 값 갯수별 지정법

  • padding: n n n n; 상 우 하 좌 순서로 각각
  • padding: n n n; 상, 좌우동일, 하 순서
  • padding: n n; 상하, 좌우 서로 동일
  • padding: n; 전부 동일
