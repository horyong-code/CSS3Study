display - 요소가 화면에 나타나는 방식 지정

  • display: inline; inline 성격으로 지정
  • display: block; block 성격으로 지정
  • display: inline-block; inline + block 성격으로 지정
  • display: none; 화면에서 사라짐

display: none;은 영역과 요소가 같이 사라짐. (레이아웃의 변화 있음) visibility: hidden;은 영역 남기고 요소만 사라짐. (레이아웃의 변화 없음)

opacity: n; 요소의 불투명도 지정 0 ~ 1 사이의 값으로 변경하며 0일 때 완전 투명, 1일 때 불투명.
